# mycourse.work

White-label, multi-tenant course platform built with Nuxt and Cloudflare.

**mycourse.work** powers online training for organisations across New Zealand and beyond — from drone pilot certification to workplace onboarding and professional development.

## What we do

- AI-powered course creation with real-time generation
- Per-tenant branding, authentication, and content isolation
- Mermaid diagrams, quizzes, progress tracking, and certificates
- Deployed on Cloudflare (Workers, Pages, D1, R2)
